# Mascot Assets

Original IP direction: flat dark video-clipping owl mascot for Video Summary.

## Canonical flat SVGs

Use these first in the app. They are transparent, lightweight, and closer to the selected flat concept direction.

- `flat/logo.svg` - main flat mascot logo.
- `flat/actions-sheet.svg` - 8-pose flat action sheet.
- `flat/actions/` - individual flat action SVGs.

## Ready-to-use transparent PNGs

These PNGs were generated during exploration and are kept for reference. Prefer the flat SVGs above for product use.

- `mascot-logo.png` - high-resolution main mascot logo.
- `mascot-logo-512.png` - square 512 px app/UI logo.
- `mascot-logo-256.png` - square 256 px compact UI logo.
- `mascot-actions-sheet.png` - 8-pose transparent action sheet.

## Source files

The `source/` folder keeps the chroma-key source images used to produce the transparent PNGs. App code should use the transparent PNGs above, not the chroma-key sources.
