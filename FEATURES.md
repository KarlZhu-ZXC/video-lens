# V1 功能规格

## 产品形态

Tampermonkey 在 Bilibili 和 YouTube 视频页注入 Shadow DOM 面板，包含摘要、视频洞察、一图流和设置四个 Tab。

## 悬浮入口

- 页面刷新后默认显示可拖动悬浮按钮。
- 点击后展开摘要 Tab，并立即开始缓存优先的文本总结。
- 已有匹配摘要时直接展示缓存，不调用文本模型。
- 浏览器全屏播放时隐藏悬浮按钮和面板。

## 摘要

- 读取当前视频信息和字幕源。
- 按界面设置选择中文、英文或翻译字幕。
- 短字幕直接总结，长字幕分块后合并。
- 流式渲染正文和低干扰模型思考。
- 支持复制、Markdown 导出、重新生成、清理当前缓存。
- 缓存隔离平台、视频、Prompt、文本模型、总结语言和字幕源。

## 视频洞察

- 基于当前摘要和字幕回答问题。
- 保留有限轮次上下文。
- 支持流式 Markdown 回答和模型思考显示。

## 一图流

一图流使用本地固定提示词拼接文本摘要，然后直接调用图片模型：

```text
根据以下视频内容总结，生成一张信息可视化的精美配图，风格清晰美观，适合作为视频总结的封面图：

{summary}
```

- 不调用文本模型生成图片 Prompt。
- 不生成或解析结构化 JSON。
- 不使用 HTML/CSS 或 DOM 卡片合成。
- 不提供文字卡、背景图、纯图片等模式选择。
- 支持图片缓存、预览、重新生成和 PNG 下载。

## 设置

文本模型和图片模型各自配置：

- Base URL
- API Key
- 模型
- 连通性测试

设置页不提供供应商选择、供应商模型预设、请求模式或一图流生成模式。文本与图片配置彼此独立。

其他设置包括：

- 界面语言
- 字幕获取及总结语言
- YouTube 字幕策略和可选官方 API 凭证
- 面板位置和拖拽宽度

## 视频源

- Bilibili 视频与列表页面，保持 `upName` 兼容边界。
- YouTube watch/shorts，支持页面字幕、youtubei fallback、翻译字幕和 JSON3/XML 解析。
- SPA 切换视频时重新读取信息，并对 YouTube 短暂旧数据进行重试。

## 缓存与安全

- 配置优先存入 Tampermonkey storage。
- 无 Tampermonkey storage 时不把 API Key 写入页面 localStorage。
- 摘要和图片结果可分别复用，并可一键清空。
- Markdown 使用 DOM 安全子集渲染，不直接注入未处理 HTML。
