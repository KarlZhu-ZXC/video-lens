## 产品形态

Tampermonkey 在 Bilibili 和 YouTube 视频页注入 Shadow DOM 面板，采用极致简约的单栏全高设计，包含摘要对话和设置。

## 悬浮入口

- 页面刷新后默认显示可拖动悬浮按钮。
- 点击后展开摘要 Tab，并立即开始缓存优先的文本总结。
- 已有匹配摘要时直接展示缓存，不调用文本模型。
- 浏览器全屏播放时隐藏悬浮按钮和面板。

## 摘要

- 读取当前视频信息和字幕源。
- 按界面设置选择中文、英文或翻译字幕。
- 短字幕直接总结，长字幕分块后合并。
- 流式渲染正文；对于 DeepSeek 等具备深度思考过程的模型，在生成期间能够流式渲染 reasoning，完成后折叠并显示耗时。
- 支持复制、Markdown 导出、重新生成、清理当前缓存。
- 缓存隔离平台、视频、Prompt、文本模型、总结语言和字幕源。

## 连续对话交互

- 摘要生成后，在内容下方提供输入框进行连续追问。
- 基于当前摘要和字幕回答问题，保留有限轮次上下文。
- 支持流式 Markdown 回答。
- 在对话交互中能够根据意图使用固定的提示词模板调用图片模型生成精美配图：

```text
根据以下视频内容总结，生成一张信息可视化的精美配图，风格清晰美观，适合作为视频总结的封面图：

{summary}
```

## 设置

文本模型和 API 图片模式各自配置：

- Base URL
- API Key
- 模型
- 连通性测试

设置页不提供供应商选择、供应商模型预设或请求模式。文本与图片配置彼此独立。

图片设置还提供 `ChatGPT 网页生图（实验）`：配置一个 ChatGPT Project 根页，由同一 userscript 为每次任务新建独立聊天、跨标签自动提交最终图片 Prompt、把生成图分块回传视频页，并在视频页确认成功消费后返回根页。回传失败时保留结果聊天；API 配置在切换模式后继续保留，网页模式失败不会自动回退 API。

API 与 ChatGPT Web 使用互斥表单组。摘要页把视频卡片、模型配置和字幕选择固定在可滚动对话上方；流式总结原位更新正文，避免滚动容器反复从顶部跳到底部。总结完成后才显示左对齐的透明工具栏；无总结时输入框禁用并由“开始总结”按钮启动生成。生图失败同时显示在对话和顶部通知中。

生成图片以内嵌预览组件展示，点击后打开 Arco 风格全屏预览。底部操作栏支持缩小、比例、放大、左右旋转、恢复适配尺寸和下载；同时支持滚轮缩放、拖动、Esc、遮罩点击和右上关闭。摘要快捷工具 tooltip 固定在按钮右下方，视频卡片统计项使用默认箭头光标。

字幕选择已合并到模型配置中的“字幕：xx”胶囊，选择新语言后等待用户主动重新生成。初始摘要和后续问答会在 reasoning 流式期间展开 `Thinking`，完成后折叠并显示思考耗时；Agent 消息不再显示吉祥物头像。最新完成回复下提供三个本地固定快捷意图，点击等同于发送对应文字。

长字幕分段摘要使用并发上限 2，保持原字幕顺序；分段正文不进入 UI，空分段重试一次，不完整整体合并再重试一次，仍异常则报错。空摘要和请求等待提示在摘要区居中。模型配置胶囊更紧凑，字幕胶囊宽度为 130px，配置与快捷意图文字为白色，快捷意图不再显示额外标题。视频卡片以 3×3 Description 网格展示 UP 主、粉丝、上传时间和平台可用统计；粉丝使用群组线框图标，投币使用带 B 标识的线框硬币图标，保证 15px 尺寸下可辨识。

ChatGPT Web 图片回传会等待跨标签分块全部到齐后再组装并发送消费 ACK，避免 Tampermonkey 不同 storage key 的通知乱序造成误报缺块或接收页过早跳转。完成态快捷功能以无边框、透明背景的纯图标显示。

其他设置包括：

- 界面语言
- 字幕获取及总结语言
- YouTube 字幕策略和可选官方 API 凭证
- 面板位置和拖拽宽度

## 视频源

- Bilibili 视频与列表页面，保持 `upName` 兼容边界。
- YouTube watch/shorts，支持页面字幕、youtubei fallback、翻译字幕和 JSON3/XML 解析。
- SPA 切换视频时重新读取信息，并对 YouTube 短暂旧数据进行重试。
- 获取字幕、生成摘要等任务发生异常时，顶部通知栏会直接显示原始错误；无公开字幕的视频会明确提示“当前视频没有公开字幕”。

## 缓存与安全

- 配置优先存入 Tampermonkey storage。
- 无 Tampermonkey storage 时不把 API Key 写入页面 localStorage。
- 发布构建不内嵌 API Key、OAuth Token 或私钥；本地 `.env.*` 仅供显式集成测试使用并保持 Git 忽略。
- 摘要和图片结果可分别复用，并可一键清空。
- Markdown 使用 DOM 安全子集渲染，不直接注入未处理 HTML。
