# Video Lens 发布前 UI 与安全整理设计

## 范围

本次只处理以下内容：

- 恢复摘要页字幕 Chip 的语言下拉选择。
- 摘要页左上标题显示为“片语-AI总结”。
- 缩窄右侧导航栏并同步缩小导航图标。
- 将 Summary 导航图标替换为 AI 星芒图标。
- 审计源码、配置和构建产物中的真实 API Key、Token、OAuth 凭据及密码。

不清理测试数据，不重写 Git 历史，也不删除用户运行时配置 API 凭据的能力。

## UI 设计

字幕 Chip 继续使用现有原生 `select` 和 `selectedSubtitleId` 数据流。只要存在字幕选项，
即使当前只有一个字幕源也显示下拉控件；切换选择只更新待选字幕，用户点击“重新生成”时才读取新字幕。

摘要页标题使用独立文案“片语-AI总结”，不修改悬浮入口和产品名“片语”。右侧导航栏由当前
72px 缩至约 56px，按钮和图标按比例缩小，并统一通过一个宽度变量更新面板计算，避免内容区错位。
Summary 使用与现有线性图标风格一致的 AI 星芒 SVG。

## 发布安全

对 Git 跟踪文件和最终 `dist/video-lens.user.js` 执行敏感信息模式扫描，重点检查常见 API Key、
Bearer Token、JWT、私钥和赋值形式的密码。测试占位值不作为敏感数据处理。运行时密钥继续只从
Tampermonkey storage 读取，构建产物不得包含实际保存值。

## 验证

- 增加或更新单字幕源下拉控件、标题和 AI 图标测试。
- 运行 `pnpm test`。
- 运行 `pnpm build`。
- 扫描源码和构建产物，确认没有真实凭据。
- 使用 `pnpm dev:harness` 检查导航宽度、图标和字幕 Chip 布局。
